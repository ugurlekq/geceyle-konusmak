# Geceyle Konuşmak — Online Dergi (Sayı 01)

Next.js + TailwindCSS tabanlı sanatsal dergi iskeleti.

## Kurulum
```bash
npm install
npm run dev
```
http://localhost:3000

## Yapı
- `/pages` — Kapak, Sayı 01 ve üç makale
- `/components/ArticleLayout.tsx` — Yazı düzeni
- `/styles/globals.css` — Loş, sinematik tema
- `/public/images` — Görseller
- `/public/audio` — Ses dosyaları
